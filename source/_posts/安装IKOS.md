## 安装IKOS

<!--more-->

首先将版本依赖解决。

To build and run the analyzer, you will need the following dependencies:

- A C++ compiler that supports C++14 (gcc >= 4.9.2 or clang >= 3.4)
- CMake >= 3.4.3
- GMP >= 4.3.1
- Boost >= 1.55
- Python 2 >= 2.7.3 or Python 3 >= 3.3
- SQLite >= 3.6.20
- TBB >= 2
- LLVM and Clang 14.0.x
- (Optional) APRON >= 0.9.10
- (Optional) Pygments

Most of them can be installed using your package manager.

其中gcc和CMake版本无问题，如果版本号不合适，就可以再用包管理器安装就好了。下面时安装GMP的方式。

```sh
wget https://gmplib.org/download/gmp/gmp-6.1.2.tar.xz
tar -xvf gmp-6.1.2.tar.xz
cd gmp-6.1.2/
./configure --prefix=$HOME/local/
make -j4
make install
cd ..
```

安装Boost，用包管理器安装即可。

```sh
sudo apt install libboost-all-dev
```

安装TBB，用包管理器安装即可。

```sh
sudo apt install libtbb-dev
```

安装SQLite，用包管理器安装即可。

```sh
sudo apt install sqlite3 libsqlite3-dev
```

LLVM需要自己安装。

可选APRON>=0.9.20和Pygments。我们暂时不安装，看后面需要再进行安装。
